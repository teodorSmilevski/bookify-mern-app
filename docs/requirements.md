# Requirements

This document describes the functional requirements for a simple appointment booking system intended for small business that need fast and easy scheduling without complex registration.

## User requirements:

1. As a user, I want to make a fast appointment and with a single click.
2. As a user, I want to enter my name and phone number so that i can identify myself without creating an account.
3. As a user, I want to select a date and time for an appointment.
4. As a user, I want to see a confirmation message after booking, so that i know my booking was successful.
5. As a user, I want to see my appointment date and time saved locally, in case i forgot.

## Admin requirements:

1. As an admin, I want to see all booked appointments, grouped by day.
2. As an admin, I want to delete an appointment.
3. As an admin, I want to manage appointments easily.
